How to Use Volleyball Pro

Your complete guide to tracking volleyball matches like a pro

1

Setup Your Match

Before starting the game, customize your match settings:

  • Team Names: Tap on "Team A" or "Team B" cards and type your custom team names directly in the app
  • Point Limit: Choose from 15, 25, 30 points, or set a custom limit
  • Language: Go to Settings to switch between English, Arabic, Kurdish Sorani, or Kurdish Badini
💡 Pro Tip:

Standard volleyball matches use 25 points per set (with a 2-point difference to win). Beach volleyball typically uses 21 points.

2

Start the Game

Tap the green "Start Game" button to begin tracking. The scoreboard will show:

  • Current set number
  • Live scores for both teams
  • Number of sets won by each team
  • Point limit for the current set
💡 Pro Tip:

The match status indicator shows "Match in Progress" with a blinking green dot when the game is active.

3

Track Points

Use the control buttons to manage scores:

  • +1 Button (Blue/Orange): Add a point to the respective team
  • − Button: Remove a point if you made a mistake
  • Undo: Revert the last action completely

Each point triggers a sound effect (if enabled) and vibration feedback.

💡 Pro Tip:

You can choose from 10 different sound effects in Settings to customize your scoring experience.

4

Manage Sets

When a set ends (a team reaches the point limit with a 2-point lead):

  • A winner modal will automatically appear
  • Tap "Next Set" to start a new set with scores reset to 0
  • Tap "Continue" to keep viewing the finished set
  • All completed sets appear in the "Sets History" section
💡 Pro Tip:

Use "Reset Set" to restart the current set from 0-0 without advancing the set counter.

5

Customize Settings

Access Settings from the main screen gear icon:

  • Language: Switch between 4 languages instantly
  • Sound Effects: Choose from 10 different sounds or turn off
  • Sound Toggle: Enable/disable all sound effects
  • Vibration: Enable/disable haptic feedback

Frequently Asked Questions

Does the app require internet?

No, Volleyball Pro works completely offline. All data is stored locally on your device.

Can I track multiple matches?

Yes, tap "New Match" to start a fresh game at any time. Previous match data is cleared when you start a new match.

What happens if I close the app during a match?

Currently, match progress is not saved between sessions. We recommend completing your match in one session.

Can I change team names during a match?

Team names are set before starting the game. To change names, go back to the main screen and edit them before starting a new match.

Is my data shared with anyone?

No. Volleyball Pro does not collect, store, or share any data. Everything stays on your device.
